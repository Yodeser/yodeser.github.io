【计算机专业学生福利】各个公司的学生优惠活动

发表于 Jul 31 2016   |   分类于 welfare   |     |   阅读次数

  • 微软Dreamspark

1. 获取Dreamspark资格

如果你是学生，可以尝试获取微软Dreamspark资格。Dreamspark是微软的一项计划，可以为学生免费提供原来需要收费的一些软件工具和服务，帮助学生进行专业的开发。例如免费的Azure云，免费的Windows Server 2012等等。

首先需要有一个微软账户，然后进入Dreamspark页面 并登陆，然后点击下面的验证账户，之后会出现几种验证方法。最容易成功的是以下两种：

使用EDU邮箱验证
使用国际学生证验证

edu邮箱需要你的学校分配给你一个edu域名的邮箱，而且并不是所有学校的edu邮箱都可以通过微软的验证。比如我的学校就只给研究生开放edu邮箱的注册。你可以向你在其他大学的同学借，他们可能会有。
也可以使用国际学生证验证，但是前提是需要有一张国际学生证。如果没有的话可以去办一张，费用大概是100块左右吧，有效期是一年。通过使用国际学生证上面的序列号，就可以验证Dreamspark。另外，Dreamspark的验证有效期是一年（到期可以重新验证），所以如果验证成功之后，就赶快开发使用它的福利吧。

2. 利用Dreamspark获取免费的Azure云

Dreamspark验证通过之后，就可以申请免费的Azure云服务了。可以无限期（其实是Dreamspark的有效期，也就是一年，到期需要重新验证）使用Azure云的服务，具体的使用量由于我也是刚刚申请成功的，所以不太清楚。
首先到Portal Azure网站，注册一个订阅。然后选择第一项Dreamspark，它会弹出一个注册页面。如果是普通注册的话，需要验证手机号和信用卡。对于Dreamspark验证，不需要信用卡，只要验证手机号就可以了。但是它这里没有中国大陆的手机号验证，因此比较难办。我是去淘宝买了一次手机验证，商家会把验证码给你发过来。申请成功之后需要等一段时间才能成功登录Portal Azure。然后在该网站里就可以部署查看你自己的Web应用了。
当然还可能需要安装相应的SDK。目前Azure云支持大部分的web应用类型，ASP.NET，java，node.js，python，ruby，php都支持。所以Azure云还是很强大的。

3. 使用Dreamspark提供的其他软件

除了免费的Azure云，Dreamspark还提供了其他的一些软件下载。常用的软件有Visual Studio 2015 Community，针对学生的Xamarin，SQL Server 2014 SP1，Windows Server 2012 R2 with Update这些，完整的软件列表可以在微软Dreamspark页面找到。

  • 阿里云学生优惠

阿里云同样也有学生优惠，这个的要求很简单，只需要两点。

支付宝实名认证。
注册学信网。

满足这两点就自动获得了阿里云学生优惠的资格。优惠的资格到了毕业就结束了，所以赶快利用起来吧。拥有了资格，然后就可以购买指定的服务器，一个月只需要9.9元。服务器的配置是1核1G内存1M带宽，系统盘的容量和所选的操作系统有关， linux是20G，Windows是40G。

有了云服务器之后就可以部署自己的应用了。可以安装IIS然后部署ASP.NET的网站，也可以安装tomcat和jdk来部署java应用，也可以装数据库软件做数据库服务器。具体怎么用随心所欲，只要保证服务器不要资源占用太高就可以。

  • 腾讯云学生优惠

腾讯云的学生优惠需要两步：

实名认证。上传自己举身份证的照片，并等待审核。
学生认证。上传自己带学生证的照片，等待审核。

审核完成之后，就可以获得腾讯云的优惠券，折算下来云服务器的价格只需要每个月1元。优惠券是按月发放的，如果三个月不使用腾讯云服务将会停止发放优惠券，所以需要注意一下。

如果使用1元的服务器，那么配置同样是1核1G内存1M带宽，系统盘是Linux 8G，Windows 50G，根据自己的需要选择系统。优惠期限同样是等到毕业就结束了。所以赶紧行动起来吧，动手来打自己的网站吧！

  • Jetbrains学生优惠

Jetbrains公司的学生优惠和微软公司的差不多，有国际学生证或者edu邮箱的都可以申请。申请成功之后就可以免费使用Jetbrains公司提供的所有软件和功能。申请链接在这里：https://www.jetbrains.com/shop/eform/students 。申请完成之后需要使用Jetbrains发给你的验证邮件中的地址申请一个Jetbrains账号，使用这个账号，就可以免费试用其提供的各种优秀的工具。账号的使用期也就是你毕业的时候。

  • Github 学生开发包

这也是一个重量级的学生优惠。需求和上面的差不多，到Github教育即可看到。可以使用的东西如下，可以看到优惠十分丰厚：

Atom，一个优秀的编辑器。
AWS教育，一个国外的在线教育平台。15美元优惠。
Bitnami，一键安装云应用。一年的使用期。
CrowdFlower，一个大数据平台。
DigitalOcean，一个云服务提供商，可以搭自己的网站哦。
dnsimple，一个DNS服务提供商。两年的个人DNS服务。
Github。学生身份内托管5个私有仓库。
Hackhands，一个活跃的编程帮助。
Visual Studio，微软的功能强大的IDE。免费使用Visual Studio 2013。
Namecheap，域名注册服务。一个一年的免费me域名和一年的免费SSL证书。
Unreal Engine，虚幻引擎。学生有效期内免费试用虚幻引擎。

还有好几个我就不列出了，请自行到Github网站查看。

可惜的是由于各种原因国内的edu.cn的邮箱貌似是被拉黑名单了。所有使用edu.cn邮箱的申请请求都会被秒拒。所以只能使用上传证件的方式。上传证件的时候需要入学证明、学生证或者其他可以证明你学生身份的东西，而且必须有日期确保你是在校学生。等待人工验证通过之后就可以了。
